Location Tracking & Accuracy

The accuracy depends on GPS signal and network coverage, typically within a few meters in open areas. If the GPS device is turned off, the last known location will remain visible until the device is powered on again. You can track multiple Kawach devices simultaneously, with each child displayed on the map.

User Profile Management

To manage user profiles:

  • Add new users through the Add New User button in the Users section
  • Edit child information in the Users tab
  • Ensure app has photo gallery permissions for profile pictures.
  • All information is stored locally for privacy.

Troubleshooting

Common issues and solutions:

  • Missing notifications: Check app permissions and battery settings
  • Map marker issues: Verify user information in Users section
  • Location accuracy: Ensure device has clear sky view
  • Device offline: Check if the GPS device is powered on and has network coverage
